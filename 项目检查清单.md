### 数据库创建
- [ ] 创建数据库 herb_platform
  ```sql
  CREATE DATABASE herb_platform
  DEFAULT CHARACTER SET utf8mb4
  COLLATE utf8mb4_unicode_ci;
  ```

- [ ] 执行 schema.sql 创建表结构
  ```bash
  mysql -u root -p herb_platform < database/schema.sql
  ```

- [ ] 执行 data.sql 导入初始数据
  ```bash
  mysql -u root -p herb_platform < database/data.sql
  ```

### 数据库验证
- [ ] 验证表创建成功
  ```sql
  USE herb_platform;
  SHOW TABLES;  -- 应显示20张表
  ```

  ```sql
  SELECT username, user_type FROM sys_user;
  ```

  ```sql
  SELECT role_name, role_code FROM sys_role;
  ```
