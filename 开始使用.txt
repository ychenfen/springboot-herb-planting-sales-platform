🚀 快速启动（推荐）
==========================================

步骤1: 初始化数据库（5分钟）
------------------------------------------
mysql -u root -p
source database/schema.sql
source database/data.sql

验证：
USE herb_platform;
SHOW TABLES;  -- 应显示20张表
SELECT * FROM sys_user;  -- 应显示3个用户

步骤2: 配置后端连接（2分钟）
------------------------------------------
编辑文件：backend/src/main/resources/application-dev.yml
修改数据库用户名和密码、Redis配置

步骤3: 启动后端（2分钟）
------------------------------------------
cd backend
mvn spring-boot:run

访问：
- http://localhost:8080/api/doc.html（API文档）
- http://localhost:8080/api/druid（数据库监控）

步骤4: 启动前端（2分钟）
------------------------------------------
cd frontend
npm install
npm run dev

访问：
- http://localhost:5173
