## 快速开始

### 1. 初始化数据库

```bash
# 登录MySQL
mysql -u root -p

# 执行初始化脚本
source database/schema.sql
source database/data.sql
```

### 2. 配置后端

编辑 `backend/src/main/resources/application-dev.yml`，修改数据库和Redis连接信息。

### 3. 启动后端

```bash
cd backend
mvn spring-boot:run
```

### 4. 启动前端

```bash
cd frontend
npm install
npm run dev
```
